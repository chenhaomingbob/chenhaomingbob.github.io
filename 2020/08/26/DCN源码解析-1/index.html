Deform_conv.py

里面包含 DeformConvFunction 和 ModulatedDeformConvFunction 两个继承了torch.autograd.Function的function

torch.autogrid.Function

Descrption : Records operation history and defines formulas for differentiating ops

Two Static function need overrideen

backward (ctx : Any , *grad_outputs: Any ) -> Any

It must accept a context ctx as the first argument, followed by as many outputs did forward() return, and it should return as many tensors, as there were inputs to forward()

forward (ctx : Any , *args : Any , **kwargs : Any ) -> Any

这边重点关注 ModulatedDeformConvFunction

看之前先了解里面用到的Pytorch相关方法

  1. torch.Tensor.new_empty(size, dtype=None, device=None, requires_grad=False) –> Tensor

official : Returns a Tensor of size size filled with uninitialized data. By default, the returned Tensor has the same torch.dtype and torch.device as this tensor.

重点在于创建的tensor是为初始化，并且dtype和device 于原本的tensor相同。

  1. torch.autograd.function._ContextMethodMixin.save_for_backward ( * tensors)

official : Saves given tensors for a future call to backward(). This should be called at most once, and only from inside the forward() method

这边就是将需要梯度计算的tensor，保存到ctx中，然后在backward中调用.
